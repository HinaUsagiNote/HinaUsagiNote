# Unstack()
- stack() 반대로 index를 컬럼으로 변환한다.
- 매개변수 
    - level: multi-index일 경우 컬럼으로 만들 레벨을 지정한다. 기본값은 -1 로 가장 안쪽의 index를 이동시킨다.

```python
import pandas as pd
import numpy as np

df = pd.read_csv('data/state_fruit.csv', index_col=0)
s1 = df.stack()
s1 = df.stack()
```
