```python
#5.
fruits = "사과 복숭아 귤 배"
```


```python
# 위 fruits에 "수박"이 있는지 확인하는 코드를 작성하시오.
print('수박' in fruits) # fruits 안에 '수박'이 있는지
```

    False
    


```python
# fruits 안에 '수박'이 있으면 '수박있음' 없으면 '수박없음' 출력
print('수박있음' if '수박' in fruits else '수박없음')
```

    수박없음
